## 🛠️ How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/yashvanthbalaji/Java-program.git
   ```

2. Navigate to the folder:
   ```bash
   cd Java-program
   ```

3. Compile and run any Java file:
   ```bash
   javac FileName.java
   java FileName
   ```
